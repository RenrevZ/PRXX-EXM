### INSTALLATION GUIDE

> NOTE! THIS APPLICATION ONLY TESTED IN MYSQL AS DATABASE

- INSERT YOUR DB CREDENTIALS ON .ENV
- AFTER PROVIDING THE DETAILS NEEDED RUN
-
> php artisan migrate --seed
-
- THEN RUN 
-

> npm run dev
> php artisan serve

-
- the application should run now
